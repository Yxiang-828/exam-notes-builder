# Welcome to Notes 2 Website! 🚀

This is a **live demo** of the `notes-2-website` HTML interface!

As you can see, the sidebar on the left dynamically generates an interactive, collapsible table of contents from any Markdown (`.md`) files you place inside the `staging/` directory!

### Features you can test right now:
1. **Smart Search (`Ctrl+K`)**: Try pressing `Ctrl+K` and searching for the word "Notes". It will instantly pop open the fuzzy search and highlight every match!
2. **Session Deletion**: Try clicking the little Trash Can icon next to this chapter name on the left sidebar. It instantly wipes the chapter from the UI cleanly.
3. **Markdown Rendering**: Try writing Python code blocks, adding Mermaid diagrams, putting in images, or complex tables. The offline parser instantly supports it all smoothly!

## How did this page get here?
This page was perfectly generated by creating a simple folder structure on the computer before clicking 'Publish', like this:

```
staging/
└── Example_Demo/
    └── Welcome.md   <--- (This very file you are reading!)
```
